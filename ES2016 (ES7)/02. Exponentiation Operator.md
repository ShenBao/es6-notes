# Exponentiation Operator(幂运算符)

如何实现一个数的求幂运算？

```js
function pow(x, y) {
  let res = 1;
  for (let i = 0; i < y; i++) {
    res *= x;
  }
  return res;
}

console.log(pow(2, 10));
// 1024
```

除了自己封装函数来实现，也可是使用 `Math.pow()`。

`Math.pow()` 函数返回基数（base）的指数（exponent）次幂。

```js
console.log(Math.pow(2, 10)); // 1024
```

在 ES7 可以这样写了：

```js
console.log(2 ** 10); // 1024
```

注：`幂运算符的两个 * 号之间不能出现空格，否则语法会报错`
