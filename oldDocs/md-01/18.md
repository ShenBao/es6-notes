async await

1. 不依赖于外部的runner了——统一、性能

```js
let readData = async () => {
    let data1 = await $.ajax({ url: 'data/1.txt', dataType: 'json' });
    let data2 = await $.ajax({ url: 'data/2.txt', dataType: 'json' });
    let data3 = await $.ajax({ url: 'data/3.txt', dataType: 'json' });

    console.log(data1, data2, data3);
}

readData();
```
